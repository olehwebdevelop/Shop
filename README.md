# Modern E-Commerce Store

Modern responsive online store concept focused on clean UI, smooth user experience, and performance-focused frontend development.

## Features

- Fully responsive layout
- Modern product showcase sections
- Interactive product cards
- Smooth animations and hover effects
- Clean navigation system
- Optimized spacing and typography
- Mobile-friendly interface
- Lightweight and fast-loading structure

## Technologies

- HTML5
- CSS3
- JavaScript

## Sections

- Hero
- Products
- Categories
- Pricing
- About
- Contact

## Goals

This project was created to explore modern e-commerce frontend development, responsive layouts, and interactive UI design while maintaining clean and scalable code.
